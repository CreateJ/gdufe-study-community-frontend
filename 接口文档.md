/login

```javascript
{
```

# 添加评论功能(/comment/add/{discussPostId})

- 若添加成功，则跳转到：/discuss/detail/{discussPostId}，考虑放在msg中

```json
{
	"code": 200,
	"msg": "/discuss/detail/{discussPostId}"
}
```

- 添加失败：

```json
{
	"code": 2001,
	"msg": "添加评论失败，请重新尝试！"
}
```
